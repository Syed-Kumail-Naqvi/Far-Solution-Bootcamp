Changes made in day-1/index.html and day-1/style.css:

1. Completed all sections:
   - Hero section (existing)

2. Ensured all sections are fully responsive:
   - Mobile ( <768px ): single column, stacked layout
   - Tablet (768px - 1024px): two columns for services, products
   - Desktop ( >1024px ): three columns for services, products

3. Added proper spacing, margins, paddings throughout.

4. Verified all links use href="#". (No actual destinations yet)

5. Tested in Chrome DevTools device emulator (iPhone, iPad, desktop).

6. Final polish:
   - Consistent button styles (hover effects, border-radius)
   - Card shadows and hover lift effects
   - Smooth scrolling (html { scroll-behavior: smooth; })
   - Accessibility: alt texts, semantic headings, focus states.

Key Learnings:
- A complete landing page needs: Header/Nav, Hero, Services, Products, About, Contact, Footer.
- Responsive design is not just media queries — it's also about content priority.
- Always test on actual devices or emulators.
- Keep CSS organized (comments, grouping related styles).
