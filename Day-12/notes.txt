Day 12 - Functions & Scope

Topics Covered:
- Function Declaration (function name() {})
- Function Expression (const name = function() {})
- Arrow Functions (const name = () => {})
- Parameters and Arguments
- Return Statement
- Scope: Global vs Local
- Block Scope (let, const)

Key Learnings:
- Use function declarations for named reusable functions.
- Arrow functions are shorter and don't bind their own 'this'.
- Variables declared with 'var' are function-scoped, 'let' and 'const' are block-scoped.
- Functions can return values that can be used later.
- Good practice: each function should do one thing only.
